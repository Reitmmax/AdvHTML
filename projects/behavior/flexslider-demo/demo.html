How to implement the Basic Slider

The following instructions assume you have copied this file and included css, images, and js folders to your final project folder.

  1. To the <head> element of the page that will contain the slider, add the following after the link to your project .css file: 
    <link rel="stylesheet" href="css/flexslider.css">
<script src="js/modernizr.js"></script>
  2. Add a <div> with unordered list of images for the slider inside either the <main> element or page <header> element. Note you must assign flexslider class to the div element and slides class to the unordered list element as shown in the example below. 
    <div class="flexslider">
  <ul class="slides">
    <li><img src="images/slide1.jpg">
    <p class="flex-caption">Caption text</p></li>
    <li><img src="images/slide2.jpg"></li>
    <li><img src="images/slide3.jpg"></li>
  </ul>
</div>
  3. Optionally, to display a caption under a slide, add a <p> with assigned flex-caption class after the <img> element as shown in the example above.
  4. Add the following to the bottom of the page, just before the closing </body> tag: 
    <!-- jQuery -->
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1/jquery.min.js">
</script>
<script>window.jQuery || document.write('<script 
src="js/libs/jquery-1.7.min.js">\x3C/script>')</script>

<!-- FlexSlider -->
<script defer src="js/jquery.flexslider.js"></script>
<script type="text/javascript">
  $(window).load(function(){
    $('.flexslider').flexslider({	  
      animation: "slide", 
      start: function(slider){ 
        $('body').removeClass('loading');  
      }
    });
  });
</script>

<!-- Optional FlexSlider Additions -->
<script src="js/jquery.easing.js"></script>
<script src="js/jquery.mousewheel.js"></script>
<script defer src="js/demo.js"></script>
  5. Once the slider is in place and successfully working, replace the demo files in the images folder with your own images. Note all images should be scaled to approximately the same dimensions.
